Welcome to my webpage!

This webpage is a HTML webpage!

That means that this webpage was made by using the code language HTML!

HTML is a great language because you can code webpages and do anything you want with the your webpage!

HTML stands for Hyper Text Markup Language

HTML also goes along with 2 other coding languages called CSS and JavaScript.

CSS is how you style your webpage and color it. CSS's full form is Cascading Style Sheets.

JavaScript is how you can make buttons and other things to interact with your webpage.

There is also a component called the console. The console is for testing purposes.

You can go to the console of any webpage by right clicking on the webpage and clicking inspect.

To print something onto the console in HTML you need to create a script tag.

To create a script tag, you need to make 2 angular brackets like this <>. Inside them, write script.

Then you can print something on the console by creating a variable like this: var = you can write anything you want here.

Then to print it onto the console, you can do: console.log(var).

Another way you can print words in the console is: console.log("You can write whatever you want to print in the console here.")
